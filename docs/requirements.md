# 需求分析（A1_聽寫練習）

## 產品願景與角色
- 家長：建立/管理題庫、設定練習規則、查看孩子表現。
- 孩子：在網頁上聆聽題目、手寫作答、即時得到回饋。
- 系統：生成語音、管理練習流程、儲存結果並維持資料安全。

## 功能需求
1. 題庫建立
   - 家長可建立字詞/短句清單，可標註語言、注音/拼音、釋義、年級、標籤。
   - 匯入/匯出 CSV；版本留痕（家長改動可追溯）。
2. 出題設定
   - 家長從題庫勾選或隨機抽取指定數量字詞交由 AI 生成練習題。
   - 練習形式：聽寫（語音播放詞語，或句中隱去一字），可設定播報次數、語速、停頓。
   - 題目隨機化：題序打亂、同詞多語音樣本。
3. 孩子作答體驗
   - 觸控手寫區（平板/手機/桌機手寫板），筆畫粗細/顏色可調，橡皮擦與重寫。
   - 練習進度條、剩餘題數提示，支援鍵盤輸入作為備選。
   - 無廣告、簡潔大按鈕介面，色彩對比與可讀性符合兒童可及性。
4. 評分與回饋
   - 作答結束後，提供正確寫法展示、差異提示（筆畫或字形比對），播放正確語音。
   - 累積錯誤字詞清單，支持「錯題重練」。
5. 報表與家長面板
   - 線上紀錄：日期、題組、正確率、耗時、薄弱標籤。
   - 下載/匯出 JSON 或 CSV。
6. 系統設定
   - 家長帳號與角色權限（家長/孩子分離）。
   - 語音 TTS 選擇（本地或雲端），語速、語調。
   - 資料備份/還原，日誌與隱私設定。

## 非功能需求
- 裝置適配：桌機、平板、手機，PWA 可加到主畫面。
- 安全：帳密/OTP 登入，孩子介面鎖定；資料儲存在掛載 volume，便於備份。
- 隱私：最少化個資，作答影像/墨跡僅在本地或私有儲存。
- 音訊效能：TTS 生成可快取；播放無明顯延遲。
- 可維運性：容器化部署，設定與資料透過 volume 持久化；記錄可觀察性日誌。
- 部署優先：盡量本地可執行（GPU 支援）作為預設；雲端僅作補充。

## 初步資料實體
- user（家長/孩子）、word_entry、word_list（題庫與版本）、practice_set、practice_item、attempt（手寫結果/評分）、audio_cache、config（TTS、語速、安全）。

## AI/模型使用原則
- 優先本地化：TTS（如 Piper/Coqui）、語音辨識（Whisper.cpp/自建 ASR）、手寫辨識（PaddleOCR/HWR）本地部署。
- LLM（出題/講解）：本地可用模型優先；需高品質時使用 Google Gemini (AI Studio) 為雲端備援。
- 不傾向使用付費 Whisper API；如需雲端 ASR 以 Google Speech 為首選。
- BYOK：家長可在設定中填入 Gemini/其他 LLM Key，覆蓋系統預設；未填時使用本地模型或停用進階功能。
