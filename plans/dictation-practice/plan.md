# 聽寫練習功能計畫（A5 Card）

## 目標

提供國小學童「聽、寫、認」三合一的國字練習功能，涵蓋多種題型和遊戲化計分機制。題庫來自教育部課綱生字表（南一/康軒/翰林），支援家長自訂範圍。

---

## 一、出題範圍

### 模式 1：隨機出題
- 從 `vocabulary.json` 全庫 2,979 字中隨機選取
- 適合日常練習、綜合複習

### 模式 2：指定範圍
- **依課綱篩選**：三層下拉選單
  - 出版社（南一 / 康軒 / 翰林）
  - 年級（1-6 年級）
  - 課次（第 N 課）→ 可多選
- **家長自訂**：手動輸入生字清單（逗號或換行分隔）
- 適合期中/期末複習、隨堂小考

## 二、題型模式

### 題型 A：例句填空選擇題
- **形式**：顯示例句，關鍵字挖空，提供 4 個選項
- **資料來源**：教育部辭典即時爬取例句（同 A1 的 MOE provider）
- **生題算法**：
  1. 從範圍中選一個字
  2. 查辭典取得含該字的造詞
  3. 造句或取辭典例句
  4. 挖空目標字，製造 3 個干擾選項（同音字/形近字優先）
- **適合**：認字初級、低年級

### 題型 B：例句填空手寫題
- **形式**：顯示例句，關鍵字挖空，用 HanziWriter quiz 手寫作答
- **流程**：
  1. 顯示挖空例句
  2. 出現 HanziWriter 白框，小孩手寫
  3. HanziWriter 逐筆判定對錯
  4. 寫完揭露正確答案 + 筆順動畫
- **適合**：認字進階、中年級

### 題型 C：聽寫測驗
- **形式**：語音唸出詞語，小孩手寫
- **流程**：
  1. TTS 唸「請寫出：ㄒㄩㄝˊ ㄒㄧㄠˋ」（注音）
  2. 同時語音唸「學校」（國語）
  3. 小孩在 HanziWriter 白框逐字手寫
  4. 寫完第一字自動跳下一字
  5. 全部寫完 → 批改 + 筆順重播
- **技術**：
  - `speechSynthesis` Web API（免費、離線可用）
  - 語速可調（慢/中/快）
  - 可重聽（點按喇叭圖示）
- **適合**：聽寫考試模擬、高年級

### 題型 D：描寫練習
- **形式**：顯示一個字，先看筆順動畫，再練習描寫
- **流程**：
  1. 顯示字 + 注音 + 造詞
  2. 自動播放筆順動畫
  3. 進入 HanziWriter quiz 描寫模式
  4. 可重複描寫多次（3 次為佳）
  5. 每次描寫計分
- **適合**：寫字練習、低年級

### 擴充性設計
```typescript
interface QuizMode {
  id: string
  label: string
  icon: string
  generate(chars: string[], count: number): Promise<Question[]>
  renderQuestion(q: Question): React.ReactNode
  evaluate(q: Question, answer: unknown): Score
}
```
新增題型只需實作此介面，不需改動出題框架。

## 四、計分與遊戲化

### 4.1 基本計分
| 事件 | 得分 |
|------|------|
| 選擇題答對 | +1 |
| 手寫一次過（無提示） | +3 |
| 手寫有提示後完成 | +1 |
| 聽寫全對 | +5 |
| 聽寫部分對（按字算） | +1/字 |
| 描寫完成 | +1 |

### 4.2 連擊系統（Combo）
- 連續答對累計 combo 計數
- combo ≥ 3：顯示 🔥 + combo 數字
- combo ≥ 5：分數 ×1.5
- combo ≥ 10：分數 ×2 + 特效
- 答錯 combo 歸零

### 4.3 即時反饋
- **答對**：綠色閃爍 + 音效「叮！」+ "+N" 飄字動畫
- **答錯**：紅色搖晃 + 顯示正確答案 + 筆順動畫教學
- **連擊**：火焰特效 + combo 計數器放大
- **全對**：煙火特效（canvas-confetti）

### 4.4 等級徽章系統
| 累計分數 | 等級 | 徽章 |
|---------|------|------|
| 0-9 | 識字新手 | 🌱 |
| 10-29 | 小小學生 | 📖 |
| 30-59 | 造詞高手 | ✏️ |
| 60-99 | 成語達人 | 🏅 |
| 100-199 | 國語小博士 | 🎓 |
| 200+ | 文字大師 | 👑 |

等級顯示在 Header 計分板旁邊，升級時播放特效。

### 4.5 遊戲化參考
- **成語小偵探** (idiom.gh.miniasp.com)：即時判定 + combo + 激勵語
- **Duolingo**：生命值（3 條命）、每日挑戰、經驗值升級
- **Kahoot**：限時搶答、排行榜
- **打字遊戲**：速度挑戰、字從天上掉下來要接住

### 4.6 可選進階機制（未來）
- **每日挑戰**：每天 5 題，連續打卡送特效
- **錯題本**：自動收集答錯的字，下次優先出題
- **間隔重複**：根據遺忘曲線排程複習
- **家長報表**：哪些字常錯、學習進度圖表

## 五、技術架構

### 後端
```
POST /api/a5/quiz
Body: {
  mode: "random" | "curriculum" | "custom",
  publisher?: "南一版" | "康軒版" | "翰林版",
  grade?: "3年級",
  lessons?: ["第一課", "第二課"],
  customChars?: "學校花草",
  questionCount: 10,
  quizType: "choice" | "handwrite" | "dictation" | "trace"
}
Response: {
  ok: true,
  quizId: "...",
  items: [
    {
      id: "q-1",
      character: "學",
      bopomofo: "ㄒㄩㄝˊ",
      prompt: "他在＿＿上寫了名字。",  // 題型 A/B
      word: "學校",                    // 題型 C/D
      options?: ["學", "覺", "角", "較"], // 題型 A
    }
  ]
}
```

### 前端
```
A5Page
├── QuizSetup         — 出題設定（範圍/題數/題型）
├── QuizSession       — 答題進行中
│   ├── ChoiceQuestion    — 選擇題
│   ├── HandwriteQuestion — 手寫填空
│   ├── DictationQuestion — 聽寫
│   └── TraceQuestion     — 描寫
├── QuizResult        — 結果 + combo 統計
└── QuizReview        — 錯題回顧 + 筆順教學
```

### 資料流
```
vocabulary.json ──→ 後端篩選 ──→ 出題 ──→ 前端答題
                                  ↑
                    MOE 辭典爬取例句（即時）
                    Gemini 造句（fallback）
```

## 六、實作優先順序

| 階段 | 內容 | 預估工作量 |
|------|------|-----------|
| Phase 1 | 出題設定 UI + 描寫練習（題型 D） | 最輕量，HanziWriter 現成 |
| Phase 2 | 聽寫測驗（題型 C）+ TTS 整合 | 中等，TTS API 簡單 |
| Phase 3 | 例句填空選擇題（題型 A） | 需要例句資料 |
| Phase 4 | 例句填空手寫題（題型 B） | Phase 3 + HanziWriter |
| Phase 5 | 連擊系統 + 等級徽章 + 音效 | 遊戲化層 |

建議從 Phase 1（描寫練習）開始，因為 HanziWriter quiz 模式已經在 A1 驗證過可行，只需要加上出題設定和計分。
