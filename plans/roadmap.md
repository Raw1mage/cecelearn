# 希希小家教 — 功能藍圖

## 現有功能

### A1 查字詞
- 語音輸入 → Gemini AI 校正（同音字糾錯）→ 教育部辭典即時查詢
- 筆順動畫（HanziWriter）+ 描字練習（quiz 模式）
- 造詞 / 成語（教育部辭典 + Gemini fallback）
- 直式注音標註（ruby 排版）
- 完成放煙火 + 計分板

### A2 成語練習
- 成語選擇題（從詞庫出題）
- 答對計分 + 100% 放煙火
- **進行中**：教育部成語典爬蟲（1665 條），例句式出題

### A3 四則運算
- 計算機介面 → 直式運算動畫（加減乘除）
- 逐步 highlight + 進位拉下動畫
- 除法 CSS 括號 + 位數對齊

## 短期計畫（Milestone 5-6）

### M5: 成語出題引擎重構
- **題庫**：`idioms.json`（爬蟲產出，~1600 成語 + 例句）
- **出題模式**：
  1. 隨機模式 — 從全庫隨機選取
  2. 指定模式 — 家長自訂成語範圍
- **題型**：例句填空（從例句中挖空成語，4 選 1）
- **算法出題**：純演算法，不需要 AI
  - 正確答案 = 例句中的成語
  - 干擾選項 = 從題庫隨機抽 3 個不同成語
- **UI**：出題設定加下拉選單切換模式

### M6: 聽寫練習（新卡片 A5）
- **核心流程**：TTS 唸題 → 小孩手寫作答 → 自動評分
- **技術方案**：
  - 出題：`speechSynthesis` Web API（瀏覽器 TTS，免費）
  - 手寫：HanziWriter `quiz()` 串接（逐字辨識）
  - 多字聽寫：寫完一字自動跳下一字
  - 評分：HanziWriter 回報筆劃正確率
- **題庫**：
  - 短期：手動建 JSON（按年級常用字）
  - 長期：爬教科書生字表（康軒/南一/翰林）
- **出題模式**：同成語（隨機 / 指定範圍）
- **遊戲化**：連續答對 combo 特效、計分板

## 中期計畫（Milestone 7-8）

### M7: 英文單字練習（新卡片 A6）
- **聽**：TTS 唸單字 / 例句
- **寫**：canvas 手寫英文字母 → 辨識（26 字母 ML 模型或 Tesseract）
- **讀**：Web Speech API 辨識發音
- **題庫**：按年級 / 教材單元分類

### M8: 通用出題引擎（模組化）
```
QuizEngine（通用框架）
├── QuestionBank     — 題庫載入（JSON / API / 手動輸入）
├── QuestionGen      — 出題策略（隨機 / 範圍 / 難度遞進 / 錯題重練）
├── AnswerInput      — 答題介面（選擇題 / 手寫 / 語音 / 拖拉排序）
├── Scorer           — 評分邏輯 + 計分板整合
└── Effects          — 特效引擎（煙火 / 音效 / 連擊 / 等級徽章）
```
- 每個科目只需實作 `QuestionBank` + `AnswerInput`
- 共用出題策略、計分、特效

## 長期願景

### 教材進度分級
- 爬取各版本教科書進度表（康軒/南一/翰林）
- 按年級 / 學期 / 單元組織題庫
- 家長可選「跟著學校進度」自動出題

### 學習歷程追蹤
- 錯題記錄 + 弱點分析
- 間隔重複（spaced repetition）
- 學習曲線視覺化

### AI 輔助
- 根據答錯模式自動調整難度
- 個人化出題（強化弱項）
- AI 生成例句 / 解釋

## 技術架構

| 層級 | 技術 | 說明 |
|------|------|------|
| 前端 | React + Vite + TypeScript | SPA，HanziWriter 筆順 |
| 後端 | Node.js (Bun) | API server，爬蟲代理 |

| AI | Gemini 2.5 Flash Lite | 語音校正、造詞 fallback |
| TTS | Web Speech API | 瀏覽器內建，免費 |
| 手寫 | HanziWriter quiz | 中文筆劃辨識 |
| 特效 | canvas-confetti | 慶祝動畫 |
| 部署 | Docker Compose + Nginx | Gateway:7014 |
